## ⚠️ 关于控制台警告

您可能会在浏览器控制台看到一些警告信息，这些都是**非关键性警告**，不影响功能：

### 1. Passive Event Listener 警告
```
[Violation] Added non-passive event listener to a scroll-blocking 'touchstart' event
```
- **原因**: Ionic 框架的触摸事件处理
- **影响**: 仅性能提示
- **结论**: ✅ 可以忽略

### 2. Favicon 图标尺寸警告
```
Error while trying to use the following icon from the Manifest
```
- **原因**: manifest.json 中的图标尺寸与实际文件不匹配
- **状态**: ✅ 已修复
- **操作**: 刷新浏览器页面（Ctrl+Shift+R 或 Cmd+Shift+R）
- **结论**: ✅ 警告将消失

### 3. ARIA Hidden 警告
```
Blocked aria-hidden on an element because its descendant retained focus
```
- **原因**: Ionic 路由切换时的焦点管理
- **影响**: 仅可访问性警告
- **结论**: ✅ 可以忽略

**这些警告都是 Ionic 框架的已知行为，不会影响您的使用体验。**
