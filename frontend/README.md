# Ionic Angular 前端

## 安装步骤

```bash
# 安装依赖
npm install

# 启动开发服务器
ionic serve

# 构建移动应用
ionic capacitor build ios
ionic capacitor build android
```

## 功能页面

- **刷题页面** (`question.page`) - 答题、查看解析、场景案例
- **分析页面** (`analysis.page`) - 弱点诊断、通过率预测
- **复习页面** - 基于记忆曲线的智能复习

## 移动端优化

- 响应式设计，适配各种屏幕
- 触摸友好的大按钮
- 离线缓存支持
- 原生应用体验
