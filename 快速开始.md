# 🚀 TOGAF Master - 快速开始指南

欢迎使用 TOGAF Master 刷题平台！本指南将帮助您快速上手。

---

## 📋 前提条件

确保您的系统已安装：
- ✅ PHP 8.1+
- ✅ MySQL 8.0+
- ✅ Node.js 16+
- ✅ Composer
- ✅ npm

---

## 🎯 5分钟快速启动

### 步骤 1: 启动服务器

```bash
# 在项目根目录执行
./START_SERVERS.sh
```

这个脚本会自动启动：
- 后端服务器（端口 9527）
- 前端服务器（端口 8100）

### 步骤 2: 访问应用

打开浏览器访问：
```
http://localhost:8100
```

### 步骤 3: 开始使用

1. 在首页选择测验包（Level 1 或 Level 2）
2. 点击"开始学习"
3. 开始答题！

---

## 🎓 使用流程

### 1. 选择测验包

首页显示两个测验包：
- **TOGAF Level 1 Foundation** - 基础级认证
- **TOGAF Level 2 Practitioner** - 实践级认证

点击任一测验包进入学习路径。

### 2. 查看学习单元

学习路径页面显示：
- 📚 学习单元列表
- 📊 学习进度
- 🎯 学习成果

点击"开始学习"进入答题。

### 3. 答题

答题页面功能：
- ❓ 显示题目和选项
- ⏱️ 计时功能
- ✅ 提交答案
- 📖 查看详细解析

### 4. 查看解析

提交答案后：
- ✅ 显示正确答案
- 📝 详细解析说明
- 💡 相关知识点
- ➡️ 继续下一题

## ❓ 常见问题

### Q1: 需要注册登录吗？
**A**: 不需要！本平台完全免费，无需注册登录即可使用。

### Q2: 题目会重复吗？
**A**: 系统会随机推送题目，可能会重复。这有助于巩固知识。

### Q3: 可以查看历史答题记录吗？
**A**: 当前版本暂不支持。未来可能会添加用户系统。

### Q4: 题目数量够吗？
**A**: 当前有15道题用于测试。后续会持续增加题目。

### Q5: 支持移动端吗？
**A**: 支持！使用 Ionic 框架开发，可在手机浏览器访问。

## 🔍 故障排除

### 问题 1: 无法访问前端
```bash
# 检查前端是否运行
lsof -i :8100

# 如果没有运行，手动启动
cd frontend
ionic serve
```

### 问题 2: API 请求失败
```bash
# 检查后端是否运行
lsof -i :9527

# 如果没有运行，手动启动
cd backend
php artisan serve --port=9527
```

### 问题 3: CORS 错误
**解决**: 已配置支持端口 8100, 9528, 4200。如果仍有问题，重启后端服务器。

### 问题 4: 数据库连接失败
```bash
# 检查 MySQL 是否运行
lsof -i :8889

# 检查配置
cat backend/.env | grep DB_
```

更多问题请查看 **故障排除指南.md**
